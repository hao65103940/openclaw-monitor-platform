### 1. 实时日志流（WebSocket）

**现状：**
- ❌ 只能查看历史日志
- ❌ 需要手动刷新
- ❌ 运行中任务的日志无法实时查看

**需求：**
- ✅ WebSocket 实时推送日志
- ✅ 自动追加新日志
- ✅ 支持暂停/恢复

**实现方案：**
```javascript
// 后端：Socket.IO
io.on('connection', (socket) => {
  socket.on('subscribe:logs', (sessionId) => {
    // 监听日志文件变化
    tailFile(`/tmp/openclaw/${sessionId}.log`, (line) => {
      socket.emit('log:new', parseLog(line));
    });
  });
});

// 前端：接收日志
socket.on('log:new', (log) => {
  setLogs(prev => [...prev, log]);
});
```

### 2. 错误监控告警

**现状：**
- ❌ 失败任务只能手动发现
- ❌ 没有告警通知
- ❌ 无法及时了解异常

**需求：**
- ✅ 失败检测（连续失败 N 次）
- ✅ 多渠道通知（飞书/企微/邮件）
- ✅ 告警阈值配置

**实现方案：**
```javascript
// 错误检测
function checkFailures() {
  const recentFailures = sessions.filter(s => 
    s.status === 'failed' && 
    Date.now() - s.completedAt < 3600000 // 1 小时内
  );
  
  if (recentFailures.length >= 3) {
    sendAlert({
      type: 'high_failure_rate',
      count: recentFailures.length,
      sessions: recentFailures
    });
  }
}
```

- ✅ 超时任务检测
- ✅ 资源使用监控

**实现方案：**
```javascript
// 心跳检测
setInterval(() => {
  agents.forEach(agent => {
    const lastHeartbeat = Date.now() - agent.lastActivity;
    if (lastHeartbeat > 300000) { // 5 分钟无活动
      markAgentUnhealthy(agent.id);
    }
  });
}, 60000);
```
